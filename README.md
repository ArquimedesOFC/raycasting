# raycasting
Introdução ao Raycasting

Bem-vindo ao repositório do projeto Introdução ao Raycasting. Este projeto é uma exploração inicial do conceito de raycasting, uma técnica de renderização gráfica utilizada para criar efeitos de perspectiva em 3D em um ambiente 2D.

Visão Geral

Raycasting é uma técnica onde raios são lançados a partir de um ponto (como a posição do jogador) em direções diferentes para determinar o que está visível na cena. Esta abordagem é utilizada em muitos jogos clássicos de estilo "dungeon crawler" e também serve como base para entender conceitos mais avançados de renderização 3D.

Objetivos do Projeto

Introduzir conceitos básicos de raycasting:

Como raios são lançados e como colidem com objetos.

Determinação de distância e interseções.

Renderização básica baseada em distância.

Criar uma visualização simples:

Representar paredes e objetos em um ambiente 2D.

Simular uma perspectiva 3D em um espaço 2D.

Entender as bases da renderização 3D:

Explorar como técnicas simples podem ser escaladas para aplicações mais complexas.
